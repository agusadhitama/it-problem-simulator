## 💻 IT Problem Simulator

> A creative Python based IT troubleshooting simulator with GUI & CLI hybrid system.  
> Built for learning, simulation, and portfolio showcase.

Created by me **Agus Satria Adhitama**

---

## 🚀 Overview

**IT Problem Simulator** adalah aplikasi berbasis Python yang digunakan untuk mensimulasikan berbagai masalah IT yang sering terjadi di dunia kerja.

Project ini dibuat untuk:
- Melatih skill troubleshooting IT
- Simulasi kondisi real sebagai IT Support
- Menjadi portfolio project yang unik & kreatif

GUI dibuat menggunakan **Tkinter**, library bawaan Python untuk membuat aplikasi desktop. :contentReference[oaicite:0]{index=0}

---

## ⚡ Features

- 🎨 GUI modern (dark mode style)
- 💻 CLI Mode (terminal simulation)
- 🧠 Training Mode (latihan troubleshooting)
- 🔄 Random Error Generator
- ⚠️ Simulasi berbagai masalah IT:
  - Internet Error
  - Printer Error
  - Application Crash
  - Disk Full
  - Memory Error
  - CPU Overload
  - Network Fluctuation
- 📝 Logging otomatis ke file
- 🎭 Animasi loading & interaktif

## ⚙️ Installation
1. Clone Repository
```bash
git clone https://github.com/agusadhitama/it-problem-simulator.git
cd it-problem-simulator
```
2. Requirements

Tidak membutuhkan library tambahan.

Pastikan:

Python 3.x sudah terinstall

---

## ▶️ Usage
Jalankan GUI
```bash
python main.py
```
CLI Mode
- Bisa diakses dari tombol GUI
- Atau langsung dari terminal

---

## 📦 Convert ke EXE (Optional)

Gunakan PyInstaller:
```bash
pip install pyinstaller
pyinstaller --onefile --windowed main.py
```

Hasil .exe ada di folder:

```bash
dist/
```
